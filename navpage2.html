Contact Us

Got a scholarship tip, a broken link to report, or just want to say hi? I'd love to hear from you. This site is a one-person project built to help students find funding they might not know exists — so any contribution, correction, or kind word genuinely goes a long way. Reach out through any of the options below.

Ways to Reach Me

Email

The best way to report a broken scholarship link or suggest a new one to add. I check it regularly — shoot me a message and I'll get back to you when I can.

Instagram

Send a DM if you'd prefer a more casual conversation. Always open to hearing from students. Account is private but just send a message, if it's just a request it'll be done quickly!

GitHub

Want to see how this site was built, or spot something in the code worth fixing? The full project is open on GitHub — feel free to browse, open an issue, or fork it.
